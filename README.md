# Chatpdf
## Installation
- Firstly, you need to get GOOGLE_API_KEY in https://aistudio.google.com/app/apikey. Then, you will set GOOGLE_API_KEY in file .env.
- Git clone this repo: https://github.com/trthminh/chatpdf.git
- After that, you will install libraries using run
  ```
  pip install -r requirements.txt
  ```
## Run app
- You can run this app by
```
streamlit run main.py
```
